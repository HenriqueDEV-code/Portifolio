Sistema de Gestão de Imóveis (CaptaWeb)

Olá! Este é o meu projeto Sistema de Captação de Imóveis (CaptaWeb). Criei um esboço inicial do sistema e estou desenvolvendo uma nova versão, aprimorando-o continuamente. Para o desenvolvimento utilizei C# como backend e Windows Forms como frontend. O banco de dados utilizado foi o MSSQL e implementei integração com API de consulta de CEP para facilitar o cadastro de endereços.
