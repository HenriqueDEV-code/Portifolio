Sistema de Gestão de HortiFruti (PIM)

Este é um sistema básico desenvolvido em C, com o diferencial de possuir uma interface visual criada utilizando Windows.h para um design mais amigável. O objetivo do sistema é auxiliar no controle de estoque, geração de relatórios, gerenciamento de contas, PDVs e também implementar um sistema de doação de alimentos próximos do vencimento, evitando desperdícios e promovendo o aproveitamento dos produtos.
